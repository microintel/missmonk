Matrix & Types

A matrix is a rectangular array of numbers arranged in rows (m) and columns (n), enclosed in brackets. Written as an m × n matrix.

A = [aij] where i = 1,2,...,m (rows) and j = 1,2,...,n (cols)
Order = m × n  |  Element at row i, col j = aij

EXAMPLE — 3×3 MATRIX

[
a₁₁a₁₂a₁₃ a₂₁a₂₂a₂₃ a₃₁a₃₂a₃₃
]
↑ Row 1  → a₁₁, a₁₂, a₁₃
↑ Row 2  → a₂₁, a₂₂, a₂₃
↑ Row 3  → a₃₁, a₃₂, a₃₃

Algebra of Matrices

➕

Addition / Subtraction

Only possible when orders are identical. Add/subtract corresponding elements.

A ± B → (a±b)ᵢⱼ = aᵢⱼ ± bᵢⱼ
Same order required!
✖️

Scalar Multiplication

Multiply every element of the matrix by the scalar k.

kA → (kA)ᵢⱼ = k · aᵢⱼ
Order stays the same.
🔢

Matrix Multiplication

A is m×p, B is p×n → result is m×n. Inner dimensions must match!

(AB)ᵢⱼ = Σ aᵢₖ · bₖⱼ
k = 1 to p

Think of each output element as a dot product: take row i of A and column j of B, multiply element-by-element, then sum. The result sits at position (i,j) in the product matrix. Cols of A MUST equal Rows of B

1.3

Determinants & Singular Matrices

The determinant is a special scalar value computed from a square matrix. It encodes critical information: whether a matrix is invertible, the volume scaling of the transformation, and more.

2×2 Determinant

det(A) = |A| = ad − bc

A = [[a, b], [c, d]]
|A| = a·d − b·c
EXAMPLE

A = [[3, 2], [1, 4]]

|A| = (3)(4) − (2)(1) = 12 − 2 = 10

3×3 Determinant (Cofactor Expansion)

|A| = a₁₁(a₂₂a₃₃−a₂₃a₃₂)
     − a₁₂(a₂₁a₃₃−a₂₃a₃₁)
     + a₁₃(a₂₁a₃₂−a₂₂a₃₁)

Expand along Row 1
✦ STEP-BY-STEP — 3×3 DETERMINANT

Find |A| for A = [[2,1,3],[0,4,1],[5,2,6]]

1

Expand along Row 1: |A| = 2·|M₁₁| − 1·|M₁₂| + 3·|M₁₃|

2

Minor M₁₁ = [[4,1],[2,6]] → (4·6)−(1·2) = 24−2 = 22

3

Minor M₁₂ = [[0,1],[5,6]] → (0·6)−(1·5) = 0−5 = −5

4

Minor M₁₃ = [[0,4],[5,2]] → (0·2)−(4·5) = 0−20 = −20

5

|A| = 2(22) − 1(−5) + 3(−20) = 44 + 5 − 60 = −11

A matrix is SINGULAR if  |A| = 0  (No inverse exists!)
A matrix is NON-SINGULAR if  |A| ≠ 0  (Inverse exists!)

Singular ↔ Rows/Cols are linearly dependent ↔ System has no unique solution
🔺 Sign PatternThe cofactor sign pattern for 3×3: + − + / − + − / + − +. Use this to get the right sign for each minor.
♻️ Row OperationsSwapping two rows flips the sign of the determinant. Multiplying a row by k multiplies det by k.
🎯 Quick Singular CheckIf two rows (or cols) are identical OR proportional → det = 0 → matrix is singular.
1.4

Cramer's Rule

Cramer's Rule uses determinants to solve systems of linear equations. For n equations in n unknowns with a non-singular coefficient matrix.

2 Variables

a₁x + b₁y = c₁
a₂x + b₂y = c₂

D = |A| = a₁b₂ − a₂b₁
Dx = c₁b₂ − c₂b₁
Dy = a₁c₂ − a₂c₁

x = Dx/D    y = Dy/D

3 Variables

AX = B   (Ax+By+Cz = constants)

D = det(A)
Dx = det(Aₓ) — replace col 1 with B
Dy = det(A_y) — replace col 2 with B
Dz = det(A_z) — replace col 3 with B

x=Dx/D   y=Dy/D   z=Dz/D
✦ WORKED EXAMPLE

Solve: 2x + y = 7   &   x + 3y = 11

1

Set up: A = [[2,1],[1,3]], B = [7,11]

2

D = det(A) = (2)(3) − (1)(1) = 6 − 1 = 5

3

Dx = det([[7,1],[11,3]]) = (7)(3)−(1)(11) = 21−11 = 10

4

Dy = det([[2,7],[1,11]]) = (2)(11)−(7)(1) = 22−7 = 15

5

x = Dx/D = 10/5 = 2     y = Dy/D = 15/5 = 3

✓ Verify: 2(2)+1(3) = 7 ✓    1(2)+3(3) = 11 ✓

🧠 Replace Column Trick

To find Dx: replace column 1 (x-coefficients) with the constants. For Dy: replace column 2. For Dz: replace column 3. The denominator D is always the main determinant — if D = 0, Cramer's Rule doesn't apply (no unique solution).

1.5

Adjoint & Inverse of a Matrix

🔄

Cofactor Matrix

Replace each element aᵢⱼ with its cofactor Cᵢⱼ = (−1)^(i+j) × Mᵢⱼ where Mᵢⱼ is the minor.

↕️

Adjoint (Adj A)

The transpose of the cofactor matrix. Swap rows and columns of the cofactor matrix.

Adj(A) = (Cofactor matrix)ᵀ
⁻¹

Inverse A⁻¹

Only exists when |A| ≠ 0 (non-singular matrix). Satisfies A × A⁻¹ = I.

A⁻¹ = Adj(A) / |A|

2×2 Inverse Shortcut

For A = [[a,b],[c,d]],   |A| = ad − bc

A⁻¹ = (1/|A|) × [[d, −b], [−c, a]]

Simply: swap diagonal, negate off-diagonal, divide by det!
✦ WORKED EXAMPLE — 2×2 INVERSE

Find A⁻¹ for A = [[4, 7], [2, 6]]

1

Find det: |A| = (4)(6) − (7)(2) = 24 − 14 = 10

2

Swap diagonal, negate off-diagonal: [[6, −7], [−2, 4]]

3

Divide by det: A⁻¹ = (1/10) × [[6,−7],[−2,4]] = [[0.6,−0.7],[−0.2,0.4]]

4

Verify: A × A⁻¹ = I = [[1,0],[0,1]] ✓

✦ 3×3 INVERSE — ADJOINT METHOD

Steps for A⁻¹ (3×3)

1

Find |A| — confirm it's non-zero.

2

Find all 9 minors Mᵢⱼ (2×2 determinants after removing row i, col j).

3

Build cofactor matrix: Cᵢⱼ = (−1)^(i+j) × Mᵢⱼ. Apply + − + / − + − / + − + signs.

4

Transpose cofactor matrix to get Adj(A).

5

A⁻¹ = Adj(A) / |A| — divide every element by the determinant.

✅ Verify Your InverseAlways check: A × A⁻¹ should give the identity matrix I. If any element is off, recheck your cofactors.
❌ Singular = No InverseIf det = 0, stop! No inverse exists. The system either has no solution or infinitely many.
🔗 Adjoint IdentityA × Adj(A) = |A| × I. This is the key identity behind the inverse formula.

Characteristic Equation & Eigenvalues

An eigenvalue λ (lambda) of square matrix A is a scalar such that multiplying A by a special vector (eigenvector) just scales it — doesn't change direction.

DEFINITIONA·v = λ·v
→
REARRANGE(A − λI)·v = 0
→
NON-TRIVIAL SOLdet(A − λI) = 0
→
CHARACTERISTIC EQSolve for λ
For A = [[a, b], [c, d]]:

det(A − λI) = det([[a−λ, b], [c, d−λ]]) = 0

(a−λ)(d−λ) − bc = 0

λ² − (a+d)λ + (ad−bc) = 0
λ² − trace(A)·λ + det(A) = 0
✦ WORKED EXAMPLE — EIGENVALUES

Find eigenvalues of A = [[4, 1], [2, 3]]

1

Characteristic equation: det(A − λI) = 0

2

A − λI = [[4−λ, 1], [2, 3−λ]]

3

det = (4−λ)(3−λ) − (1)(2) = 0

4

Expand: 12 − 4λ − 3λ + λ² − 2 = 0
λ² − 7λ + 10 = 0

5

Factor: (λ−5)(λ−2) = 0 → λ₁ = 5, λ₂ = 2

✓ Check: λ₁ + λ₂ = 7 = trace(A) = 4+3 ✓
✓ Check: λ₁ × λ₂ = 10 = det(A) = (4)(3)−(1)(2) ✓
Sum of eigenvalues = trace(A)
Product of eigenvalues = det(A)
Eigenvalues of A² = λ²
Eigenvalues of A⁻¹ = 1/λ
Eigenvalues of kA = kλ
For each eigenvalue λ, solve (A−λI)v = 0. The solution vector v is the eigenvector. Geometrically, A only stretches/shrinks v by factor λ — no rotation.
Yes! If the discriminant of the characteristic equation is negative, eigenvalues are complex (a ± bi). For real symmetric matrices, eigenvalues are always real.
If λ = 0 is an eigenvalue, then det(A) = 0 (since product of eigenvalues = det). This means A is singular and not invertible!

Quick Reference Summary

Matrix: Rectangular array m×n | Element = aᵢⱼ
A+B: aᵢⱼ + bᵢⱼ (same order) | kA: k·aᵢⱼ
AB: (AB)ᵢⱼ = Σaᵢₖbₖⱼ (cols A = rows B)
|A|₂ₓ₂: ad−bc | |A|₃ₓ₃: cofactor expansion
Singular: |A|=0 | Non-Singular: |A|≠0
Cramer: x=Dₓ/D, y=D_y/D, z=D_z/D
A⁻¹: Adj(A)/|A| | Adj(A): transpose of cofactor matrix
Eigenvalues: det(A−λI)=0 → λ²−tr(A)λ+det(A)=0
tr(A)=λ₁+λ₂ | det(A)=λ₁·λ₂
