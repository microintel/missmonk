Matrix - Complete Study Guide

Definition + Types + Examples

What is a Matrix?

A matrix is a rectangular arrangement of numbers, symbols, or expressions arranged in rows and columns.

Row: Horizontal arrangement

Column: Vertical arrangement

⎡ 1 2 3 ⎤
⎣ 4 5 6 ⎦

Order of Matrix

If a matrix has m rows and n columns, its order is m × n.

2 × 3 matrix:
⎡ 1 2 3 ⎤
⎣ 4 5 6 ⎦

Types of Matrices

1. Row Matrix

Only one row

[ 1 2 3 4 ]
2. Column Matrix

Only one column

⎡ 1 ⎤
⎢ 2 ⎥
⎣ 3 ⎦
3. Square Matrix

Rows = Columns

⎡ 1 2 ⎤
⎣ 3 4 ⎦
4. Zero Matrix

All elements are zero

⎡ 0 0 ⎤
⎣ 0 0 ⎦
5. Diagonal Matrix

Only diagonal elements are non-zero

⎡ 5 0 0 ⎤
⎢ 0 3 0 ⎥
⎣ 0 0 8 ⎦
6. Identity Matrix

Diagonal = 1, others = 0

⎡ 1 0 0 ⎤
⎢ 0 1 0 ⎥
⎣ 0 0 1 ⎦
7. Scalar Matrix

Identity matrix multiplied by a constant

⎡ 4 0 0 ⎤
⎢ 0 4 0 ⎥
⎣ 0 0 4 ⎦
8. Symmetric Matrix

A = Aᵀ (transpose is same)

⎡ 1 2 3 ⎤
⎢ 2 5 6 ⎥
⎣ 3 6 9 ⎦
9. Skew-Symmetric Matrix

Aᵀ = -A (diagonal = 0)

⎡ 0 2 -1 ⎤
⎢ -2 0 4 ⎥
⎣ 1 -4 0 ⎦
10. Upper Triangular Matrix

All elements below diagonal are zero

⎡ 1 2 3 ⎤
⎢ 0 5 6 ⎥
⎣ 0 0 9 ⎦
11. Lower Triangular Matrix

All elements above diagonal are zero

⎡ 1 0 0 ⎤
⎢ 4 5 0 ⎥
⎣ 7 8 9 ⎦
12. Singular Matrix

Determinant = 0 (no inverse)

⎡ 2 4 ⎤
⎣ 1 2 ⎦
13. Non-Singular Matrix

Determinant ≠ 0 (has inverse)

⎡ 2 3 ⎤
⎣ 1 4 ⎦

Summary

A matrix is a structured way of organizing numbers. Different types are used in engineering, physics, computer science, and data systems.
