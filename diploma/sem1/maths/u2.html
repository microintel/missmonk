2.1

Slope of a Straight Line

The slope (or gradient) of a straight line measures its steepness — how much it rises or falls per unit of horizontal distance. It is usually denoted by the letter m.

Formula — Slope from Two Points m = y₂ − y₁x₂ − x₁   where x₁ ≠ x₂
Formula — Slope from Angle of Inclination (θ) m = tan(θ)   where 0° ≤ θ < 180°, θ ≠ 90°

Positive Slope ↗

Line rises left to right. m > 0. Acute angle with positive x-axis.

Negative Slope ↘

Line falls left to right. m < 0. Obtuse angle with positive x-axis.

Zero Slope →

Horizontal line. m = 0. Parallel to x-axis.

Undefined Slope ↕

Vertical line. m is undefined (division by zero). Parallel to y-axis.

Example 2.1

Find the slope of the line joining A(3, 2) and B(7, 10).

m = (y₂ − y₁) / (x₂ − x₁) = (10 − 2) / (7 − 3) = 8 / 4 = 2 ∴ Slope m = 2 ✓
Example 2.2

A line makes an angle of 135° with the positive x-axis. Find its slope.

m = tan(θ) = tan(135°) = tan(180° − 45°) = −tan(45°) = −1 ∴ Slope m = −1 ✓

2.2

Intercepts of a Straight Line

An intercept is the point where a line crosses one of the coordinate axes.

X-Intercept Set y = 0 and solve for x. The line crosses the x-axis at (a, 0).
Y-Intercept Set x = 0 and solve for y. The line crosses the y-axis at (0, b).

The x-intercept is denoted a and the y-intercept is denoted b.

Example 2.3

Find the x-intercept and y-intercept of the line 3x + 4y = 12.

X-intercept: Set y = 0 3x + 4(0) = 12 → 3x = 12 → x = 4 ∴ X-intercept = 4, point (4, 0) Y-intercept: Set x = 0 3(0) + 4y = 12 → 4y = 12 → y = 3 ∴ Y-intercept = 3, point (0, 3)

2.3

Intercept Form of a Straight Line

When we know where a line crosses both axes, the Intercept Form gives the most direct equation. Here a is the x-intercept and b is the y-intercept (both ≠ 0).

Intercept Form xa + yb = 1

Verify: when x = a, y = 0 ✓   when x = 0, y = b ✓

Example 2.4

Write the equation of a line with x-intercept = 5 and y-intercept = −4.

Using x/a + y/b = 1 with a = 5, b = −4: x/5 + y/(−4) = 1 x/5 − y/4 = 1 Multiply throughout by 20: 4x − 5y = 20 ∴ Equation: 4x − 5y = 20 ✓
Example 2.5

The intercepts of a line on x and y axes are equal. If the line passes through (3, 3), find the equation.

Equal intercepts ⟹ a = b Equation: x/a + y/a = 1 → x + y = a Line passes through (3,3): 3 + 3 = a → a = 6 ∴ Equation: x + y = 6 ✓

2.4

Slope-Intercept Form

The most widely used form. When you know the slope m and the y-intercept c (where the line crosses the y-axis), the equation is:

Slope-Intercept Form y = mx + c    (m = slope, c = y-intercept)

From any equation Ax + By + C = 0, rearrange to y = mx + c to read off slope and y-intercept immediately.

Example 2.6

Write the equation of a line with slope 3 and y-intercept −2.

y = mx + c y = 3x + (−2) y = 3x − 2 ∴ Equation: y = 3x − 2 ✓
Example 2.7

Find the slope and y-intercept of 4x − 2y + 8 = 0.

Rearranging to y = mx + c: 4x − 2y + 8 = 0 −2y = −4x − 8 y = 2x + 4 ∴ Slope m = 2, y-intercept c = 4 ✓

2.5

Slope-Point Form

When you know the slope m and one point (x₁, y₁) on the line:

Slope-Point Form (Point-Slope Form) y − y₁ = m(x − x₁)

Derivation: Any other point (x, y) on the line satisfies slope = (y − y₁)/(x − x₁) = m, giving y − y₁ = m(x − x₁).

Example 2.8

Find the equation of a line with slope 2 passing through (3, −1).

y − y₁ = m(x − x₁) y − (−1) = 2(x − 3) y + 1 = 2x − 6 y = 2x − 7 ∴ Equation: y = 2x − 7 (or 2x − y − 7 = 0) ✓

2.6

Two-Point Form

When you know two points (x₁, y₁) and (x₂, y₂), the equation is derived by first computing slope, then using slope-point form:

Two-Point Form y − y₁y₂ − y₁ = x − x₁x₂ − x₁
Equivalent Determinant Form |x   y   1|
|x₁ y₁ 1| = 0
|x₂ y₂ 1|
Example 2.9

Find the equation of a line through (1, 3) and (4, 9).

Step 1: Slope m = (9 − 3)/(4 − 1) = 6/3 = 2 Step 2: y − y₁ = m(x − x₁) y − 3 = 2(x − 1) y − 3 = 2x − 2 y = 2x + 1 ∴ Equation: y = 2x + 1 (or 2x − y + 1 = 0) ✓ Verify with (4,9): 2(4) + 1 = 9 ✓

2.7

General Form of a Straight Line

Every straight line (including vertical lines) can be expressed in the General Form:

General Form Ax + By + C = 0   where A, B are not both zero

From general form, we can extract all key information:

PropertyFormula from Ax + By + C = 0
Slope (m)m = −AB   (B ≠ 0)
X-intercept (a)a = −CA   (A ≠ 0)
Y-intercept (b)b = −CB   (B ≠ 0)
Example 2.10

For the line 5x − 3y + 15 = 0, find: (a) slope (b) x-intercept (c) y-intercept.

Here A = 5, B = −3, C = 15 (a) Slope m = −A/B = −5/(−3) = 5/3 (b) X-intercept a = −C/A = −15/5 = −3 → crosses x-axis at (−3, 0) (c) Y-intercept b = −C/B = −15/(−3) = 5 → crosses y-axis at (0, 5) ✓

2.8

Angle Between Two Lines & Conditions

Let two lines have slopes m₁ and m₂. The angle θ between them satisfies:

Angle Between Two Lines tan(θ) = |m₁ − m₂1 + m₁m₂|   (1 + m₁m₂ ≠ 0)

The acute angle is given by taking the positive value. The obtuse angle = 180° − θ.

⟺ Parallel Lines

m₁ = m₂

Two lines are parallel if and only if their slopes are equal.

⊥ Perpendicular Lines

m₁ · m₂ = −1

Two lines are perpendicular if and only if the product of their slopes is −1.

Example 2.11

Find the acute angle between lines y = 2x + 3 and y = −x + 5.

m₁ = 2, m₂ = −1 tan(θ) = |(m₁ − m₂)/(1 + m₁m₂)| = |(2 − (−1))/(1 + 2×(−1))| = |3/(1 − 2)| = |3/(−1)| = 3 θ = arctan(3) ≈ 71.57° ✓
Example 2.12

Are the lines 2x − 4y + 5 = 0 and x − 2y + 7 = 0 parallel, perpendicular, or neither?

Line 1: 2x − 4y + 5 = 0 → slope m₁ = −2/(−4) = 1/2 Line 2: x − 2y + 7 = 0 → slope m₂ = −1/(−2) = 1/2 m₁ = m₂ = 1/2 → Lines are PARALLEL ✓ (but C values differ, so they don't coincide)

2.9

Equation of a Line Parallel to a Given Line

A line parallel to Ax + By + C = 0 has the same A and B coefficients. It differs only in the constant term:

Parallel Line Formula Given line: Ax + By + C = 0
Parallel line: Ax + By + K = 0  (K ≠ C, K determined by given point)

Key: Same slope → same A and B. Only the constant changes. Substitute the given point to find K.

Example 2.13

Find the equation of a line parallel to 3x + 5y − 10 = 0 passing through (2, 1).

Parallel line has form: 3x + 5y + K = 0 Substitute point (2, 1): 3(2) + 5(1) + K = 0 6 + 5 + K = 0 K = −11 ∴ Equation: 3x + 5y − 11 = 0 ✓ Check: slope of 3x+5y−10=0 is −3/5 slope of 3x+5y−11=0 is −3/5 ✓ (equal → parallel)

2.10

Equation of a Line Perpendicular to a Given Line

A line perpendicular to Ax + By + C = 0 swaps and negates the A and B coefficients:

Perpendicular Line Formula Given line: Ax + By + C = 0
Perp line: Bx − Ay + K = 0  (K determined by given point)

Why it works: Slope of given line = −A/B. Slope of perp line = B/A. Product = (−A/B)(B/A) = −1 ✓

Example 2.14

Find the equation of a line perpendicular to 2x − 3y + 6 = 0 passing through (4, 1).

Given: 2x − 3y + 6 = 0 (A=2, B=−3) Perp line form: Bx − Ay + K = 0 → (−3)x − (2)y + K = 0 → 3x + 2y + K = 0 [multiplied by −1] Substitute (4, 1): 3(4) + 2(1) + K = 0 12 + 2 + K = 0 K = −14 ∴ Equation: 3x + 2y − 14 = 0 ✓ Verify: slope of given = −2/(−3) = 2/3 slope of perp = −3/2 Product = (2/3)(−3/2) = −1 ✓
