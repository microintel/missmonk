Number Systems Tutorial

Decimal, Binary, Octal, Hexadecimal & Conversions

1. Introduction to Number Systems

A number system is a way to represent numbers using symbols and rules. The most common number systems are:

  • Decimal (Base 10)
  • Binary (Base 2)
  • Octal (Base 8)
  • Hexadecimal (Base 16)

2. Decimal Number System (Base 10)

Uses digits 0 to 9. It is the standard system used in daily life.

Example: 245₁₀ = 2×10² + 4×10¹ + 5×10⁰

Solved Problem: Convert 156₁₀ into binary

156 ÷ 2 = 78 r0
78 ÷ 2 = 39 r0
39 ÷ 2 = 19 r1
19 ÷ 2 = 9 r1
9 ÷ 2 = 4 r1
4 ÷ 2 = 2 r0
2 ÷ 2 = 1 r0
1 ÷ 2 = 0 r1

Answer: 10011100₂

3. Binary Number System (Base 2)

Uses only 0 and 1. Used in computers.

Example: 1011₂ = 11₁₀

Solved Problem: Convert 1101₂ to decimal

= 1×2³ + 1×2² + 0×2¹ + 1×2⁰
= 8 + 4 + 0 + 1 = 13₁₀

4. Octal Number System (Base 8)

Uses digits 0 to 7.

Example: 17₈ = 15₁₀

Solved Problem: Convert 25₈ to decimal

= 2×8¹ + 5×8⁰ = 16 + 5 = 21₁₀

5. Hexadecimal Number System (Base 16)

Uses digits 0–9 and A–F.

Example: A5₁₆ = 165₁₀

Solved Problem: Convert 2F₁₆ to decimal

= 2×16¹ + 15×16⁰ = 32 + 15 = 47₁₀
